# ND Readable Romance Master Index v1.1

## Core use

This is the **live pack map**.

Use it to answer:

- what files exist now
- which files are core drafting tools
- which files are control notes
- what order the tools depend on each other
- which file to open next
- what should be archived

This file is not a drafting tool.
It is a **control file**.

---

## Pack closure rule

A file belongs in the live canonical pack only if it is:

- present
- named correctly
- actually used by the working system
- not superseded by a cleaner replacement

## Anti-drift note

Do not keep both a superseded file and its corrected canonical replacement active in the same live pack.

That creates:

- naming split
- false choice
- stale references
- future confusion about which file is authoritative

---

## Closure test

The control layer counts as closed only when all of these are true:

- every referenced file exists
- the canonical naming line is consistent
- the master index lists real files only
- the workflow file points only to real files
- overlap notes describe the actual current state
- superseded files are marked for archive
