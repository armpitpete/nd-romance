# Templates

Clean copyable blank forms live here.

Use these when you want to start writing without editing the guidance files in `active_pack/`.

## Recommended order

1. `mini_design_blank.md`
2. `structure_spine_blank.md`
3. `arc_map_blank.md`
4. `scene_card_blank.md`
5. `beat_tracker_blank.md`

## Rule

Copy a template into your project folder before filling it in.

Do not overwrite the blank template files.
