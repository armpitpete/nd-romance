# Web Prototype

This folder contains the first static web prototype for the ND Romance pack.

## Current file

```text
web/index.html
```

## What it does

- Shows the Mini Design Sheet as a browser form.
- Autosaves draft input in browser `localStorage`.
- Generates a live Markdown preview.
- Copies Markdown to clipboard.
- Downloads the result as a `.md` file.

## What it does not do

- no login
- no database
- no backend
- no AI writing assistant
- no GitHub sync
- no Google Drive sync

## Current status

This is a local/static prototype. It can later be served through GitHub Pages.
