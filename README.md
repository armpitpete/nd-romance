# ND Romance

A working GitHub copy of the ND-readable romance planning and revision pack.

## Start here

Open these first:

```text
PROJECT_STATUS.md
FIRST_USE.md
```

`PROJECT_STATUS.md` shows the current state and next steps.
`FIRST_USE.md` gives the shortest path through the pack.

## Working source

Use this folder as the active working area:

```text
active_pack/
```

## Clean blank templates

Copy blank working forms from:

```text
templates/
```

Recommended order:

1. `templates/mini_design_blank.md`
2. `templates/structure_spine_blank.md`
3. `templates/arc_map_blank.md`
4. `templates/scene_card_blank.md`
5. `templates/beat_tracker_blank.md`
